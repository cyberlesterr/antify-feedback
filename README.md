# Antify Feedback

This is the public issue tracker for [Antify](https://antifyapp.com) — a per-app transparent proxy for macOS.

## How to Submit Feedback

- **Bug Report**: [Create a bug report](../../issues/new?template=bug_report.md)
- **Feature Request**: [Request a feature](../../issues/new?template=feature_request.md)

Before submitting, please search [existing issues](../../issues) to avoid duplicates.

## Links

- Homepage: [antifyapp.com](https://antifyapp.com)
- Install via Homebrew: `brew tap cyberlesterr/antify && brew install --cask antify`
